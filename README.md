# encode-club-weekend-project-4

tokenized ballot full-stack dApp

**note**: this repo has git sub-modules

## cloning this repo

### 3-step method

```bash
git clone <repo-url>

git submodule init

git submodule update

```

### 2-step method

```bash

git clone <repo-url>

git submodule update --init --recursive

```

### 1-step method

```bash
git clone --recurse-submodules <repo-url>
```
